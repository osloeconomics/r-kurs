Objekter og operatorer

R er et objektorientert sprk. De vanligste objekttypene er vector, matrix, data.frame og list. Vi lager objekter ved hjelp av en “venstre-pil”.

n1 <- 420

Objekter forblir lagret i minnet fram til R avsluttes, eller til man fjerner objektet manuelt ved hjelp av rm(). En av fordelene med R (sammnelignet med Stata) er at det er mulig ha mange datasett i minnet samtidig.

= brukes inni funksjoner, ikke til lage objekter:

n2 <- paste("4", "2", "0", sep = "") # "limer" sammen 4, 2 og 0

Dobbelt likhetstegn brukes for sammenligne noe (som i Stata)

n1 == as.integer(n2) 

[1] TRUE

Ellers fungerer de vanlige matematiske operatorene som forventet

(200 + 15 - 5) * 2

[1] 420

Objekttyper

Vi har alts vector, matrix, data.frame og list. matrix bestr av n eller flere vector. 

v1 <- c(1, 2, 3) # c()-funksjonen lager vektorer
v2 <- c(4, 5, 6)
matrix(c(v1, v2), nrow = 3, ncol = 2)

     [,1] [,2]
[1,]    1    4
[2,]    2    5
[3,]    3    6

data.frame er en slags matrix med kolonnenavn (men er egentlig basert p list)

class(mtcars)

[1] "data.frame"

typeof(mtcars) # typeof gir den underliggende datastrukturen

[1] "list"

Hente ut elementer fra objekter

Det finnes hovedsakelig to funksjoner for hente ut elementer av objekter, $ og []. Frstnevnte fungerer bare p data.frame og named list. Syntaksen er mtcars$kolonnenavn og mtcars[, kolonne]. I sistnevnte kan ogs rader spesifiseres.

# Alle rader, kolonnen "cyl"
mtcars$cyl 
mtcars[, "cyl"] # merk anfrselstegn her
# Rad 1 til 3, kolonne "cyl"
mtcars$cyl[1:3]
mtcars[1:3, "cyl"]
mtcars[1:3, 2] # Kan ogs bruke kolonnens plassing ("cyl" er kolonne nr. 2)

Klamme-syntaksen kan ogs brukes til lage “delmengder” av objektene:

mtcars[mtcars$cyl > 4, ] # henter ut alle rader hvor "cyl" er strre enn 4
mtcars[mtcars$cyl > 4, "cyl"] # henter ut alle verdier av "cyl" som er strre enn 4

"list"-objekter

Et list-objekt er en liste med objekter, for eksempel en liste med data.frame. list-objekter er hierarkiske og kan ha mange niver.

l1 <- list(3, 4) # `list` med ett niv
l2 <- list(list(3, 4), list(5, 6)) # `list` med to niver

Syntaksen for hente ut elementer fra en liste er litt spesiell. Dersom listen er navngitt, kan $ benyttes. Hvis ikke, brukes doble klammer, [[]]:

testlist <- list("element1" = 3, "element2" = 4) # navngitt liste 
testlist[[1]]

[1] 3

testlist$element1

[1] 3

Lister er nyttige dersom du trenger utfre samme operasjon p flere objekter. For eksempel samme regresjon p flere datasett:

mtlist <- list(mtcars, mtcars[mtcars$cyl > 4, ])
lapply(mtlist, function(x) {
    reg <- lm(mpg ~ cyl + hp + wt, data = x)
    summary(reg)
  })

Pakker

De viktigste funksjonene er integrert i R, men en del funksjoner krever at du installerer og laster inn pakker ved hjelp av library-funksjonen.

install.packages("dplyr") # Merk: anfrselstegn
library(dplyr) # Merk: ikke anfrselstegn
filter(mtcars, cyl >= 8) # Kun biler med svr motor

Det er ogs mulig benytte spesifikke funksjoner uten laste inn pakken frst:

dplyr::filter(mtcars, cyl >= 8)

Tidyverse

Vi kan ogs bruke innebygget R-funksjonalitet for filtrere data:

a <- mtcars[mtcars$cyl >= 8, ] 
b <- dplyr::filter(mtcars, cyl >= 8)
identical(a, b) # Sjekk om vi fr samme resultat

[1] TRUE

dplyr-syntaksen er bygget rundt verb, og beskriver nrmest med ord hva som skjer med dataene (p ngelsk da): “apply a filter to mtcars that keeps cars with >= 8 cylinders”.

dplyr er en del av tidyverse, en samling med pakker som er skapt for databehandling, og deler samme grammatikk, datastruktur og underliggende filosofi. Vi kommer stort sett til forbli innenfor tidyverse i dette kurset.

Vanligste funksjoner

La oss ta et steg tilbake og se p de mest brukte og nyttigste funksjonene i R. Mye av koden vi skriver vil dreie seg om databehandling. Andre oppgaver, som f.eks. regresjoner krever mindre kode, og er relativt enkelt utfre. Nedenfor ser vi p de vanligste og mest generelle innebygde funksjonene

summary(mtcars) # Oppsummering av data (ekvivalent med -summarize- i Stata)
table(mtcars$cyl) # Tabulering av data (ekv. med -tabulate- i Stata)
